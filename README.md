# AI Console Agent


### Key Installations and Set Up

* Install Go Language. You can install it <a href="https://go.dev/doc/install">here</a>

* Create your own Anthropic API Key, and set it as an environment variable, `ANTHROPIC_API_KEY`. 

---

1. Enable Dependency Tracking for Go
```bash
go mod tidy
```

2. Run AI Agent on console: 
```bash
go run main.go
```
